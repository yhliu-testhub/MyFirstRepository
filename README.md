# 图片压缩工具 - 安装和运行指南

## 前置要求

1. **安装 Python**
   - 访问 https://www.python.org/downloads/
   - 下载最新版本的 Python 3.x
   - 安装时勾选 "Add Python to PATH" 选项

2. **安装依赖库**
   ```bash
   pip install Pillow>=10.0.0
   ```

## 运行程序

1. **安装依赖后，运行程序**
   ```bash
   python main.py
   ```

## 使用说明

### 功能特点
- ✅ 支持选择单张或多张图片（jpg/png/webp）
- ✅ 可调节压缩质量（1-100%）
- ✅ 可设置最大宽度（100-4096像素）
- ✅ 实时显示压缩前后大小和压缩比例
- ✅ 批量处理多张图片
- ✅ 完成后自动打开输出文件夹

### 使用流程
1. 点击"选择图片"按钮选择要压缩的图片
2. 使用滑块或输入框设置压缩质量（默认80%）
3. 使用滑块或输入框设置最大宽度（默认1920像素）
4. 点击"选择"按钮选择输出文件夹
5. 点击"开始压缩"按钮开始批量处理
6. 等待处理完成，程序会自动打开输出文件夹

### 注意事项
- 压缩后的图片会以"原文件名_compressed.扩展名"保存
- PNG图片如果含有透明通道会自动转换为RGB模式
- 建议根据用途选择合适的压缩质量：
  - 网站/分享用：60-70%
  - 一般用途：70-80%
  - 高质量打印：85-95%
